Жиры

Жиры важны для гормонов, мозга, усвоения витаминов и защиты органов. Главное — не бояться их, а выбирать полезные источники.

💡 Рекомендации

  • Рекомендуемая норма: 0.8–1.2 г на 1 кг массы тела
  • Насыщенные жиры: до 10% от общего калоража
  • Ненасыщенные жиры: основа здорового рациона

Полезные источники жиров

📚 Интересные факты о жирах

Жиры помогают усваивать жирорастворимые витамины A, D, E и K. Без достаточного количества жиров в рационе эти витамины не смогут полноценно усвоиться организмом.

🥑 Почему жиры важны?

Жиры являются концентрированным источником энергии, обеспечивают теплоизоляцию, защищают внутренние органы и необходимы для производства гормонов.

О проекте

CaloTrack — это интеллектуальный помощник для расчета калорий и планирования питания. Проект создан для людей, которые следят за своим здоровьем и хотят достичь поставленных целей в питании.

Основные функции

  • 🧮 Расчет суточной нормы калорий по формуле Миффлина-Сан-Жеора
  • 📊 Определение оптимального соотношения белков, жиров и углеводов
  • 🍽️ Генерация персонального меню на основе ИМТ и целей
  • 📈 Прогноз изменения веса на 12 месяцев
  • 💾 Сохранение истории расчетов и избранных меню
  • 🎨 Поддержка светлой и темной темы оформления

Технологии

Проект разработан с использованием современных веб-технологий:

  • HTML5, CSS3, JavaScript (ES6+)
  • Chart.js — для визуализации данных
  • Адаптивная верстка — для корректного отображения на всех устройствах
  • LocalStorage — для хранения данных пользователя
